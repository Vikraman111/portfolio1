Snake Game

The Snake Game, built with Python's OOP principles and the Turtle graphics library, brings the classic arcade game to life in a simple, interactive format. Players control a snake that moves around the screen, growing longer as it consumes food, but must avoid colliding with walls or its own tail. The game's modular design leverages classes for key elements such as the snake, food, and game mechanics, ensuring clean, maintainable code. This project demonstrates fundamental OOP concepts, event handling, and graphical manipulation in Python, making it both educational and entertaining.

Pong Game

The Pong Game, created using Python's OOP principles and the Turtle graphics library, recreates the classic paddle and ball game with an engaging, retro feel. Players control paddles on either side of the screen to hit a bouncing ball, aiming to prevent it from passing their side. The game uses classes to encapsulate key elements, including paddles, the ball, and scoring system, providing a clean structure and easy expandability.
